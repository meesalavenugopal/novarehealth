## Project Overview

NovareHealth is a comprehensive telemedicine platform designed for the African healthcare market, enabling patients to consult with verified doctors through video consultations, receive digital prescriptions, and make payments via M-Pesa.

### Key Features

- **Patient Portal** - Find doctors, book appointments, video consultations
- **Doctor Portal** - Profile management, availability, prescriptions
- **Admin Panel** - Doctor verification, user management, analytics
- **AI-Powered Features** - Smart bio generation, fee suggestions, chat assistant
- **Video Consultations** - Integrated Zoom meetings with automatic scheduling
- **Email Notifications** - Professional HTML email templates for all workflows
- **M-Pesa Integration** - Mobile money payments (Africa-specific)
- **Prescription System** - Digital prescriptions with medicine database
- **Health Records** - Patient EHR management
